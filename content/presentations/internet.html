The Internet
in a nutshell
The Internet is computers talking to other computers
your computer sends a request to another computer
that computer sends your computer some files
those files:
  • HTML
  • CSS
  • JavaScript
HTML
HyperText Markup Language
information arranged in a structured format

    <html>
      <head>
        <title>Cool Website</title>
      </head>
      <body>
        <h1>I'm a header</h1>
        <p>I'm a paragraph</p>
        <p class="hello">And another paragraph</p>
      </body>
    </html>
  

    
I'm a header

    
I'm a paragraph

    
And another paragraph
CSS
Cascading Style Sheets
Instructions for displaying the HTML

    body {
      font-size: 10px;
    }

    h1 {
      font-size: 20px;
      background-color: #18F18F;
    }
JavaScript
interactivity && manipulation

    var h = document.getElementByClassName("hello");
    h[0].innerHTML = "wow hello";

    h[0].onclick = function(){
      console.log(h)
    }
the browser knows how to translate and display these documents
how it works
every networked device has an IP address
216.58.217.174
159.142.144.188
208.80.154.224
but that's impossible to remember
so domain names exist
ICANN manages the domain name system
Google registered google.com and connected it to 216.58.217.174
you can use HTTP to interact with the device at 216.58.217.174
HyperText Transfer Protocol
the browser handles this interaction for you
